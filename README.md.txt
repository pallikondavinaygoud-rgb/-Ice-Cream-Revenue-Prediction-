# 🍦 Ice Cream Revenue Prediction

## 📌 Project Overview

This project predicts ice cream revenue based on temperature using the Linear Regression algorithm. It demonstrates a basic Machine Learning workflow, including data loading, model training, prediction, and performance evaluation.

## 🚀 Features

* Data preprocessing
* Linear Regression model
* Revenue prediction
* Model evaluation using MAE, MAPE, and MSE

## 🛠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab
* Jupyter Notebook

## 📂 Dataset

* **Input:** Temperature (°C)
* **Output:** Revenue

## 📊 Machine Learning Algorithm

* Linear Regression

## ▶️ How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Upload `IceCreamData.csv`.
3. Run all notebook cells.
4. View the predicted revenue and evaluation metrics.

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Absolute Percentage Error (MAPE)
* Mean Squared Error (MSE)

## 📷 Output

The model predicts ice cream revenue based on temperature and evaluates its performance using standard regression metrics.
